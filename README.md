# 🚀 IntDM — International Download Manager

> A powerful, open-source download manager for macOS, inspired by Internet Download Manager (IDM).

[![macOS](https://img.shields.io/badge/macOS-14%20Sonoma+-black?style=flat-square&logo=apple)](https://www.apple.com/macos/sonoma/)
[![Swift](https://img.shields.io/badge/Swift-5.9+-F05138?style=flat-square&logo=swift&logoColor=white)](https://swift.org)
[![License: MIT](https://img.shields.io/badge/License-MIT-blue?style=flat-square)](LICENSE)

## ✨ Features (Planned)

- ⚡ **Multi-connection download** — split files into segments for parallel downloading
- ⏸️ **Pause & Resume** — resume interrupted downloads from where they left off
- 📋 **Clipboard monitoring** — auto-detect download URLs from clipboard
- 🌐 **Browser integration** — Safari, Chrome, Firefox extensions
- 🎬 **Video grabber** — detect and download streaming videos
- 📂 **Smart categorization** — auto-organize files by type
- ⏰ **Download scheduling** — schedule downloads for specific times
- 📦 **Batch download** — download multiple files at once
- 🔒 **Protocol support** — HTTP, HTTPS, FTP, SFTP

## 🏗️ Tech Stack

- **Language:** Swift 5.9+
- **UI:** SwiftUI + AppKit (hybrid)
- **Target:** macOS 14 Sonoma+
- **Architecture:** MVVM

## 🚀 Getting Started

### Prerequisites

- macOS 14 Sonoma or later
- Xcode 15+
- Swift 5.9+

### Build & Run

```bash
# Clone the repository
git clone https://github.com/YOUR_USERNAME/international-download-manager.git
cd international-download-manager

# Open in Xcode
open IntDM.xcodeproj

# Or build from command line
xcodebuild -scheme IntDM -configuration Debug build
```

## 💡 Why IntDM?

IDM (Internet Download Manager) is one of the most powerful download managers on Windows, but it has never been available on macOS. IntDM aims to bring that same power to the Mac ecosystem with a native, beautiful, and performant experience built with Swift and SwiftUI.
